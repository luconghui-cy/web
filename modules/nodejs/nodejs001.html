多版本nodejs环境安装

使用nvm管理nodejs

  • 下载nvm_noinstall.zip
  • install.cmd 以管理员身份运行 修改 root = nvm目录(c:\dev\nvm) path = nodejs目录(c:\dev\nodejs)
  • 配置环境变量 NVM_HOME(c:\dev\nvm) | NVM_SYMLINK(c:\dev\nodej) | 修改 path 添加NVM_HOME和NVM_SYMLINK
  • nvm list
  • nvm install latest | nvm install 12.18.0
  • nvm use 12.18.0

  • process 进程对象
    •             console.log(process.argv);

            argv

            第一个参数: nodejs环境的路径
