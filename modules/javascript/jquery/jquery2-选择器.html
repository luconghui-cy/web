属性选择器

            $( "[attribute|='value']" ) 要么等于给定的字符串，要么以该字符串开头，后跟连字符(-)。

            属性hreflang等于en或者以en开头后跟连字符 - 的所有a元素
            $( "a[hreflang|='en']" ).css( "border", "3px dotted green" );
        
            $( "[attribute*='value']" ) 包含子串

            属性name的值包含man字符串的所有input元素
            $( "input[name*='man']" ).val( "has man in it!" );
